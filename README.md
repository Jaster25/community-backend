<br>

## 🔍 Features

- Git Flow(Issue -> Branch -> Commit -> Push -> Pull Request -> Merge)
- REST API 설계
- 테스트 코드 작성
- JWT 인증 방식
- 전역 예외 처리, 오류 코드 정의
- 개발 환경별 Profile 그룹 분리
- 환경 변수 암호화(Jasypt 라이브러리)
- 자동 배포 환경(GitHub Actions, AWS CodeDeploy)
